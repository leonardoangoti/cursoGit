Tutorial Git

Iniciando repositorio. 

$ git init

Adicionando arquivo no indici

$ git add arquivo.txt

Comitando

$ git commit -m 'mensagem'

Adicionando repositorio remoto

$ git remote add origin url

Enviando para repositorio remoto

$ git push

Criando breanch

$ git breanch develop

Mudando de branch

$ git chekout develop

Realizando marge

$ git marge master

Criando branch git flow

$ git checkout -b feature_branch
